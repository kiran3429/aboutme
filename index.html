    Smart home

    • The smart home system enables users to control household appliances remotely via a smart device using the MQTT communication protocol, ensuring seamless real-time operation.
    • The system includes an automated temperature regulation mechanism, allowing dynamic adjustments to maintain optimal room conditions based on sensor data.
    • The system is designed to work with existing non-smart appliances by integrating 5V to 250V relays and an ESP8266 board, making it a cost-effective smart home solution.
    • A Blynk app-based UI enables intuitive control, allowing users to monitor and manage their home environment effortlessly from their smartphones.
    • The project provides a scalable architecture that can be expanded with additional sensors and features, enhancing automation capabilities for a fully smart living space.
